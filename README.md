# E2E Test Assignment

## Technologies used:

1. Node version v22.14.0
2. pnpm version 10.7.1
3. TypeScript
4. Playwright

## To run the project locally:

1. Clone the github repository
2. Run `pnpm install` - to install the project dependencies
3. Run `pnpm run pretest` - to generate test files from typescript
4. Run `pnpm run test` or `pnpm run test:ui` - to run the tests
